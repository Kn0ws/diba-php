Welcome

Description: DIBA PHPの起動確認用Intent。JSONでメッセージを返す。

  • Tags: core, example, json
  • Method: GET
  • Path: /welcome
  • Executors: WelcomeExecutor@sayWelcome
  • Response: Diba\Responses\Json

Example:

input:
  name: Taro
expected:
  message: 'Welcome to DIBA PHP!'
  time: '2025-04-19 23:13:30'
